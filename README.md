# HTTC — Governance & Fair-Play Framework

Community-created guidelines for organizing recreational table tennis play.

**Location context:** 100 Hester Street, NY 10002 · Inside MS 131 · Chinatown YMCA Beacon Program

## Disclaimer

This document was created by community volunteers to help organize recreational table tennis play. It is **not endorsed by, affiliated with, or authorized by** the YMCA of Greater New York, NYC Department of Education, MS 131, or any government agency.

These are suggested community guidelines, not enforceable rules. Participation in table tennis is a physical activity undertaken at each individual's own risk. The authors and contributors accept no liability for injuries, disputes, or damages arising from use of this framework.

## Sections

- **Overview** — Purpose, principles, stewardship context
- **House Rules** — Time rotation, equipment care, conduct guidelines
- **Queue System** — Paddle-under-table & magnetic dry-erase labels
- **Stewards** — Do/Don't guide with suggested phrases
- **Escalation** — 4-step quiet de-escalation path
- **Posted Sign** — Print-ready community guidelines
